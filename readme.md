# react-webpack-template

Simple starter for react application.

Inspired by: https://github.com/gopinav/React-TypeScript-Webpack-Starter

## Template Contains:

- TypeScript
- WebPack
- React
- Eslint
- Prettier
- Husky
- Jest
- Scss Modules

App created with it: https://github.com/xSyki/alcohol-atlas

## To install:

```bash
npx degit github:xSyki/react-webpack-template my-app
cd my-app
npm i
```

## To run:

```bash
npm start
```

## Other scripts

### Tests:

```bash
npm run test
```

### Prettier

```bash
npm run format
```

### Eslint

```bash
npm run lint
npm run lint-fix
```

## To build:

```bash
npm run build
```

Build will be in build folder.
